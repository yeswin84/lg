Claude vs GPT — 뭐가 다를까?

Claude는 GPT보다 보수적인 편이에요. 모호한 요청에는 확인 질문을 먼저 합니다. 처음엔 답답하게 느껴질 수 있지만, 복잡한 업무일수록 이 점이 장점이 됩니다.

1
claude.ai/download 접속

플랜 확인

  3. 강사에게 "완료" 손 들기

Claude Code 설치 (터미널)

Claude Code는 터미널(명령 프롬프트)에서 사용하는 AI 코딩 도구입니다.

사전 준비: Node.js 설치 확인

node --version # v18 이상이어야 함. 없으면 nodejs.org 에서 설치

Claude Code 설치

npm install -g @anthropic-ai/claude-code

설치 확인

claude --version

첫 실행 & 로그인

claude # 브라우저가 열리며 Anthropic 로그인 화면 표시 # 로그인 완료 후 터미널로 돌아옴
⚠ 주의
Windows에서는 PowerShell 또는 Git Bash를 사용하세요. 기본 cmd(명령 프롬프트)에서는 일부 기능이 제한될 수 있습니다.
✅ 지금 해보세요
  1. 터미널을 열고 claude --version 을 입력한다
  2. 버전 번호가 출력되면 성공
  3. 오류가 나면 강사에게 질문

결과는 핵심 결정사항, 액션 아이템, 다음 회의 주제 순서로 목록 형태로 작성해 주세요.

[여기에 회의 내용 붙여넣기]
✅ 지금 해보세요

  2. Claude Desktop 입력창에 붙여넣고, 오늘 강의 내용을 [회의 내용]에 넣는다
  3. 결과를 확인하고 옆 사람과 비교해본다

Projects 활용 — 맥락 유지하기

Projects는 Claude가 대화 간 맥락을 기억하게 하는 기능입니다. 같은 프로젝트 안에서 대화하면 이전 내용을 참고합니다.

Project 만드는 법

1
왼쪽 사이드바 → "Projects" 클릭

새 Project 버튼을 눌러 이름을 입력합니다. (예: "내 업무 도우미")

2
Project Instructions 작성

Claude가 이 프로젝트에서 어떻게 행동해야 하는지 설명합니다. (역할, 말투, 주요 정보)

3
Knowledge 파일 업로드 (선택)

회사 소개서, 제품 정보, 자주 쓰는 문서 등을 넣으면 Claude가 참고합니다.

📋 Project Instructions 예시
당신은 [회사명]의 마케팅 팀을 돕는 전문 AI 어시스턴트입니다.
항상 한국어로 답변하고, 간결하고 실용적인 조언을 제공해 주세요.
우리 회사의 주요 제품은 [제품명]이며, 타겟 고객은 [대상]입니다.
✅ 지금 해보세요
  1. 새 Project를 만들고 이름을 "강의 실습"으로 설정한다
  2. Project Instructions에 "한국어로만 답변해 주세요"를 입력한다
  3. 새 대화를 시작해 "안녕하세요, 자기 소개해주세요"라고 입력한다
  4. Instructions가 적용되는지 확인한다

실습: 이메일 초안 작성

📋 실습 프롬프트
당신은 전문 비즈니스 작가입니다.
아래 상황에 맞는 이메일 초안을 작성해 주세요.

상황: 거래처에 미팅 일정 확인 요청

이메일 형식: 제목, 인사, 본문, 맺음말 순서로 작성해 주세요.
✅ 지금 해보세요
  1. 프롬프트를 복사해서 Claude에 붙여넣는다
  2. 결과를 보고 마음에 드는 부분/아쉬운 부분을 메모한다
  3. Claude에게 "더 짧게 써줘" 또는 "더 정중한 표현으로 바꿔줘"라고 후속 요청한다

Claude Code 설치 & 터미널 첫 걸음

⏱ 소요 시간: 약 40분 (실습 28분)

터미널 기초 — 겁내지 마세요

터미널은 키보드로 컴퓨터에 명령하는 도구입니다. 마우스 클릭 대신 글자로 지시합니다.

명령어하는 일예시
ls (Mac) / dir (Win)현재 폴더의 파일 목록 보기ls
cd폴더 이동cd Documents
mkdir새 폴더 만들기mkdir my-project
pwd현재 위치 확인pwd
⚠ 주의
터미널에서 실수를 두려워하지 마세요. rm 명령어(파일 삭제)만 조심하면 됩니다. Claude Code가 위험한 명령은 실행 전에 확인을 요청합니다.
✅ 지금 해보세요
  1. 터미널을 열고 pwd를 입력해 현재 위치를 확인한다
  2. ls (Mac) 또는 dir (Windows)로 파일 목록을 본다
  3. mkdir claude-practice로 실습 폴더를 만든다
  4. cd claude-practice로 그 폴더로 이동한다

Claude Code 설치 & 인증

# Node.js 버전 확인 node --version # Claude Code 설치 npm install -g @anthropic-ai/claude-code # 설치 확인 claude --version

3
인증 확인
claude --version # 버전 번호가 출력되면 성공

첫 명령어 실행

Claude Code는 터미널에서 claude 명령으로 시작합니다. 대화식(interactive)과 일회성(one-shot) 두 가지 방식이 있습니다.

대화식 모드 (Interactive)

cd claude-practice claude # Claude Code REPL이 시작됨 # > 프롬프트가 표시되면 대화 시작 # 종료: /exit 또는 Ctrl+C

일회성 실행 (One-shot)

claude "안녕하세요. 지금 이 폴더에 있는 파일 목록을 알려주세요"
✅ 지금 해보세요
  1. claude-practice 폴더에서 claude를 실행한다
  2. "안녕하세요"라고 입력해 Claude Code와 첫 대화를 나눈다
  3. "현재 폴더에 hello.txt 파일을 만들어줘"라고 요청한다
  4. ls로 파일이 생성됐는지 확인한다
  5. /exit로 Claude Code를 종료한다

Claude Code 실전 실습

⏱ 소요 시간: 약 60분 (실습 45분)

프로젝트 폴더에서 Claude Code 활용

Claude Code는 폴더 안의 모든 파일을 읽고 수정할 수 있습니다. 실제 프로젝트 폴더에서 실행하는 것이 핵심입니다.

# 방법 1: 폴더로 이동 후 실행 cd my-project-folder claude # 방법 2: 해당 폴더로 이동 후 실행 cd /path/to/project && claude
⚠ 주의
Claude Code는 파일을 직접 수정합니다. 중요한 프로젝트에서는 반드시 Git으로 버전 관리를 하거나, 파일을 백업한 후 실습하세요.
